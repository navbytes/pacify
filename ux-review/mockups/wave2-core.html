Resolved decisions (from red-team):

  • One control. No master toggle; the “No proxy (direct)” row is the off state. One encoding, no ambiguity.
  • Honest model. Selecting a row switches the active proxy; the list shows it’s exclusive (fixes E4’s radio-in-disguise).
  • a11y: radiogroup + roving tabindex + aria-checked; active = check + color bar (not color alone); unselected ≠ disabled.
  • Glance: same color identity threads to the top status (Wave 1) and the toolbar icon badge (Wave 3, item 18).
